# VectorAtlas

An interactive search engine built from first principles to explore and visualize semantic retrieval end-to-end.

## Goal

Build every stage of a modern retrieval pipeline while making each step visible.

```
Document
    ↓
Extract Text
    ↓
Recursive Chunking
    ↓
Storage
    ↓
Retrieval
    ↓
Ranking
    ↓
Response
```

## The plan

Rather than treating search as a black box, VectorAtlas makes the retrieval process visible. Documents, chunks, queries, vectors, similarity scores, and ranking decisions are shown as the system processes a request.
